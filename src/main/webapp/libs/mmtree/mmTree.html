  • mmTree
    • Alternate HTML presentation format for mm format used by FreeMind mind mapping software
    • Uses MarkTree code for presentation and tree ui
    • mmTree is created by Miika Nurminen and released as Open Source (MPL 1.1/GPL/LGPL)
    • More info at MarkTree web site
    • Acknowledgements to authors of FreeMind, an excellent mind mapping tool
    • Features
      • Fully folding tree layout (opposed to HTML export in FreeMind, you can expand/collapse any node that has subnodes)
      • FreeMind-like Font styles, colours and sizes (different fonts are not supported)
        • testing font style
        • font size
        • node color
        • background color
      • FreeMind-like links, pictures,clouds and inline html.
        • internal links and arrowlinks are also supported since version 0.2
          • for example, an internal link
          • and an arrowlink
        • some inner html:
          • 11 12
            21 22
      • Anonymous nodes are cut off unless they are leaf nodes.
      • Supports Freemind Icons since version 0.2
      • Fast and intuitive MarkTree-style keyboard and/or mouse navigation with highlighted focus
        • Browse with arrows or numpad keys / * - +
        • You can fold nodes by clicking the general empy area of the intended node (no need to click tree symbols, although they work also)
      • Renders valid XHTML 1.1, provided user-entered data is valid...
      • MarkTree is tested with Mozilla 1.4, Opera 7.0 and Internet Explorer 5.0 or later versions. Might work also with other browsers/versions.
    • Installation package includes:
      • XHTML 1.1 XSL and CSS style files
      • Minimal XSL Transformer utility using Java 1.4 (you could also use any XSLT processor, like Xalan or Xerces. Works also in XSLT-enabled browsers, like Mozilla or Internet Explorer)
      • Windows batch files for converting a single file or entire directory of mm files
      • MarkTree JavaScript code and default tree pictures
